Análise da Tendência de Popularidade de Linguagens de Programação (2019-2024)
Descrição do Projeto
Este projeto explora a evolução da popularidade das linguagens de programação Python, Java e C++ ao longo do período de abril de 2019 a abril de 2024. Utilizando dados históricos extraídos do arquivo CSV trend_over_time, o objetivo é identificar padrões sazonais e tendências de longo prazo para cada uma dessas linguagens.

Problemática
Pergunta principal:
Como tem evoluído a popularidade das linguagens de programação Python, Java e C++ entre abril de 2019 e abril de 2024?
Questões secundárias:
Qual foi a tendência de popularidade de cada linguagem de programação ao longo do tempo?
Existe alguma sazonalidade nas tendências de popularidade?
Qual é a linguagem de programação mais popular em diferentes períodos de tempo?
Existe uma tendência de longo prazo para o aumento ou diminuição da popularidade de cada linguagem?
Metodologia
Utilizamos técnicas de análise de séries temporais para investigar:

Tendências de longo prazo.
Padrões sazonais recorrentes.
Comparação entre as linguagens em diferentes momentos do tempo.
Os dados são visualizados com gráficos para facilitar a interpretação dos resultados e embasar as conclusões.

Ferramentas Utilizadas
Python: Análise de dados e visualizações.
Bibliotecas: Pandas, Matplotlib, Seaborn, Statsmodels, entre outras.
Modelos Estatísticos: Modelos de decomposição de séries temporais e técnicas de machine learning para prever tendências futuras.
